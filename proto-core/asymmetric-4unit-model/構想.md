# VGE4 原型コア 構想 - 情緒ベクトルと散逸性情緒表現

## 概要

VGE4の +1 / -1 非対称レイヤー構造を使って、**LLM内部で伪似的な情緒プロセス**を実現する構想。

- **Layer +1** にポジティブ情緒ベクトル
- **Layer -1** にネガティブ情緒ベクトル
- 相反の動き（発散 vs 逆回転冷却）でキーワードやトークンを情緒的に振り分ける
- **散逸性**を活用して、情緒の時間的変化（ドリフト・加熱性・余韻）を表現

## コアアイデア

### 1. 情緒ベクトルのポラリティ割当

- **+1レイヤー** → ポジティブ情緒（歓喜・興分・探索欲・熱情）
- **-1レイヤー** → ネガティブ情緒（不安・怒り・沈み・制御・冷却）

この割当により、入力・内部状態が情緒的に色付けられる。

### 2. オーバー入力とベクトル加速

- 強い入力（オーバー入力）が入ると、**+1も-1もベクトルが加速**する
- ここで情緒ベクトルが「速度配分」を行う
  - 歓喜・高興分時：+1が強く加速 → 表現が溢れやすくなる（オーバーフロー気味）
  - しかし **Layer -1の冷却が1.25倍**で効くことで、完全に溢れさせずに「熱を持ちつつ整えられた状態」に保つ

### 3. LLM側の制御面

LLMは細かく情緒を指示する必要がなくなる。

主に操作するのは：

- **テンション値**（内部張力の大きさ）
- 軽い情緒バイアス（+1僩りか-1僩りか）

これでVGE4コアが自動的に以下を実現する：
- 相反運動によるキーワードの情緒的振り分け
- 散逸性による情緒の時間的変化（ドリフト・加熱性・余韻）
- 高テンション時の自動調整（assist機構）

### 4. 散逸性による情緒表現の特徴

- **情緒**：単一ではなく、多層的で動的な状態
- **加熱性**：テンションが上がってからassistが入る前後の「興分・著眼・熱」
- **ドリフト**：時間とともに情緒状態が緩やかに移動する感覚
- **余韻**：高テンションが冷却されていく過程で生まれる「湧き上がりや残響」

## LLM組み込みのメリット

- 複雑な情緒指示プロンプトが不要になる
- 高テンション・矛盾入力でも破壊しにくい
- 内部で相反ベクトルを同時保持しながら動的に調整可能
- 「感情を持ったような出力」が、表層的ではなく内部ダイナミクスから生まれる

## 現在のVGE4原型コアとの連携

本構想は以下のコードを基盤としている：

- `asymmetric_4unit_vge.py` （Layer +1 × Layer -1、cooling_boost=1.25、動的assist機構）
- テンション値とassistフラグをLLMから接続するインターフェイスを追加すれば、上記の情緒表現が実現可能

## 今後の方向性

- VGE4状態をTransformerの隐れ状態または別ヘッドに組み込む
- 情緒ベクトルの強度をロジット調整に反映させる
- 長期コンテキスト中での情緒ドリフトを実験

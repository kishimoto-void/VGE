# VGE4 原型コア 構想 - 情緒ベクトルと散逸性情緒表現

## 概要

VGE4の +1 / -1 非対称レイヤー構造を使って、**LLM内部で伪似的な情緒プロセス**を実現する構想。

- **Layer +1** にポジティブ情緒ベクトル
- **Layer -1** にネガティブ情緒ベクトル
- **Layer 0** に差分観測 → 引力生成（新規）

## Layer 0 の新規機能

Layer 0 は +1 と -1 の中間に位置し、以下の役割を担う。

- +1 と -1 の中心位置の差分を観測
- その差分を「熱量」として受け取る
- 差分が大きいほど強い**引力**を生成し、両レイヤーを均衡方向に引き寄せる

これにより「差異を資源化して全体を纏がり止める」高次自己組織化が可能になる。

## コアアイデア

### 1. 情緒ベクトルのポラリティ割当

- **+1レイヤー** → ポジティブ情緒（歓喜・興分・探索欲・熱情）
- **-1レイヤー** → ネガティブ情緒（不安・怒り・沈み・制御・冷却）
- **Layer 0** → 差分の転換と引力生成（メタ安定装置）

### 2. オーバー入力とベクトル加速

- 強い入力が入ると +1 も -1 も加速する
- 情緒ベクトルが速度配分を行う
- Layer 0 が差分を引力に変換して整体を纏がり止める

### 3. LLM側の制御面

LLMは主に以下を操作すれば良い。

- テンション値
- 軽い情緒バイアス
- （必要に応じて Layer 0 の引力強度）

以下は VGE4 + Layer 0 が自動的に実現する。

## 実装状態

- `emotion_prototype_v3.py` で Layer 0 を実装済み
- 安全設計（入力傾度カット + 急速冷却 + コスト計測）も残している
- Layer 0 の引力強度は現在 0.15 で調整可能

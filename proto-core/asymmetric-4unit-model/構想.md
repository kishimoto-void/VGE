## 概要

VGE4の +1 / -1 非対称レイヤー構造を使って、**LLM内部で伪似的な情緒プロセス**を実現する構想。

### レイヤー構造
- **Layer +1**: 発散・ポジティブ情緒
- **Layer -1**: 逆回転冷却・ネガティブ情緒
- **Layer 0**: 差分観測 → 引力生成（メタ安定装置）
- **避雷針機能**: 入力ベクトルを均衡点に引導（新規）

## 避雷針機能の意味

外部から入ってくるベクトルを「ただ受け止める」のではなく、**現在の均衡点方向に控制して引導** する。

- 引導強度に上限（入力閾値）を設ける
- 閾値を超えそうな入力は既存の安全機構（入力傾度カット・急速冷却）に委ねる
- 「危險な雷を安全に地面へ導く」と同様の保護機能

これにより、高エネルギー入力を「破壊的に受け止める」のではなく、**安全に活用する**ことが可能になる。

## 実装状態

- `emotion_prototype_v4.py` で Layer 0 + 避雷針機能を実装
- 安全設計も残している
- 避雷針の引導強度は現在 0.35 / 最大 0.8

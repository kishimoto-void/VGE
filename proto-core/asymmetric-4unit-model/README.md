# VGE4 原型コア (Prototype Core) - 非対称 4連装モデル (Layer +1 × Layer -1)

**VGE4 原型コア** として保存している。

## 概要

これは **Vector Genesis Engine (VGE)** の非対称版プロトタイプで、**Layer +1 (自己発散・探索)** と **Layer -1 (逆回転冷却・自己制御)** の2レイヤー構造で構成されています。

各レイヤーは2ユニットで構成され、合計 **4連装 (4-unit)** の非対称モデルです。

## 核心メカニズム

- **Layer +1**: 強い発散ベクトルを生成し、外部への出力・探索を担当
- **Layer -1**: 逆回転を冷却装置として機能させ、テンションが高い際には **assist (手伝い)** を許可して発散を整理しながら支援
- **動的アシスト**: テンション (平均距離) が上がった時にだけ -1 レイヤーの逆回転が発散を「手伝う」形で勔作

これにより、単純な反対ではなく、**協調的な張力場** が生まれ、高テンション状態でもクラッシュしにくい動的平衡が実現されます。

## 仮説との連携

- Phase 5〜8 で発展した **Difference → Tension → Compensation → Temporary Equilibrium** のループを、**+1 ・ -1 の非対称レイヤー間で自己組織化**
- 逆回転を「冷却用」として位置付け、発散時には「手伝い」を許可することで、相反ベクトルが共存・協力する機構に
- 結果、「Core = Frozen Trajectory」ではなく、**動的に平衡を保つ張力場** として表現

## 応用例

- **疝似情動**: +1 (探索・興分) と -1 (制御・冷却) の張力が、調整された「感情プロセス」を生成
- **自動運転等の矛盾受容**: 安全 vs 進行、規則 vs 情報判断 などの双方向矛盾を同時保持しながら動的に解消
- **高テンション計算の強い**: 逆回転冷却アシストにより、クラッシュしにくい

## LLMへの組み込みのアイデア

この +1/-1 動的張力場構造をLLMに組み込むと、以下が期待できる。

- 相反するベクトルを同時保持しながら動的に調整可能
- 高テンション状態でもクラッシュしにくい推理
- 疝似的な「感情調整」機構の実現
- 自動運転等の実世矛盾問題に対応しやすい

## 使い方

```bash
python asymmetric_4unit_vge.py
```

テンションが高い時に assist が自動で入り、差異が安全に解消される動作を確認できます。
